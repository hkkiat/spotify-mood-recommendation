# Mood Tracker

Mood tracker to allow users to track their moods over time, and provide encouraging messages and music recommendations based on their moods.

## Key Features

There are 5 key features:
1. User Authentication
2. Mood Logging
3. Mood Analysis
4. Recommendation
5. Discover Section
